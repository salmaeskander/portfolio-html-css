 Portfolio

## About The Project

This is my personal portfolio website where I showcase my skills, projects, and services as a **UI/UX Designer** and **Web Developer**. It includes a pricing table for freelance packages and a contact form for potential clients.

---

## Built With

- HTML5
- CSS3
- JavaScript

---

## Sections

- **Hero** — Introduction and profile photo
- **Why Choose Me** — Key strengths and value
- **Skills** — Design tools and technical skills
- **Projects** — Filterable project gallery
- **Packages** — Freelance pricing table
- **Contact** — Feedback form and contact info
